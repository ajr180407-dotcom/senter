Kebijakan Privasi

Kebijakan Privasi ini menjelaskan bagaimana aplikasi "Senter" yang dikembangkan oleh AlfiTech Studio menangani informasi pengguna.

1. Informasi yang Dikumpulkan

Aplikasi Senter tidak mengumpulkan, menyimpan, atau membagikan data pribadi pengguna. Aplikasi ini tidak memerlukan login atau akun untuk digunakan.

2. Izin Aplikasi

Aplikasi ini memerlukan izin kamera untuk mengaktifkan lampu flash (senter). Izin kamera digunakan hanya untuk fungsi senter, tidak untuk mengambil foto atau merekam video.

3. Koneksi Internet

Aplikasi ini dapat digunakan sepenuhnya offline. Fitur utama, seperti menyalakan/mematikan senter dan 10 mode kedip, bekerja tanpa koneksi internet.

4. Pihak Ketiga

Aplikasi ini tidak menggunakan layanan pihak ketiga yang mengumpulkan data pengguna.

5. Keamanan

Karena aplikasi tidak mengumpulkan data pribadi, tidak ada data yang disimpan atau diproses.

6. Perubahan Kebijakan Privasi

Kebijakan ini dapat diperbarui sewaktu-waktu. Setiap perubahan akan ditampilkan pada halaman ini agar pengguna selalu mendapatkan informasi terbaru.
